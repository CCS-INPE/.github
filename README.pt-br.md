## 🚀 Sobre o CCS

O **Centro de Controle de Satélites (CCS)** é responsável pelas operações e controle dos satélites do **INPE (Instituto Nacional de Pesquisas Espaciais)**.

O CCS atua como o núcleo operacional do sistema de controle de satélites, coordenando o monitoramento das missões, dinâmica orbital, comunicação com Estações Terrenas e planejamento operacional.

Sua infraestrutura é baseada principalmente em dois grandes sistemas de software:

* **Sistema de Tempo Real**
* **Sistema de Dinâmica Orbital**

---

## 🛰️ Operações em Tempo Real

O Sistema de Tempo Real recebe os dados de telemetria enviados pelos satélites através das Estações Terrenas durante as passagens orbitais.

Entre suas responsabilidades estão:

* Recebimento e processamento de telemetria
* Conversão de dados brutos em grandezas físicas
* Monitoramento dos subsistemas de bordo
* Visualização operacional para os controladores
* Armazenamento de dados de missão
* Geração e envio de telecomandos
* Coordenação de sessões de medida de distância e velocidade

Essas operações permitem o acompanhamento contínuo da saúde e do estado operacional dos satélites.

---

## 🌍 Dinâmica Orbital

O Sistema de Dinâmica Orbital processa os dados de rastreio recebidos das Estações Terrenas para determinar e prever a órbita dos satélites.

Suas atividades incluem:

* Determinação e propagação de órbita
* Previsão de passagens futuras
* Cálculo de visibilidade para Estações Terrenas
* Planejamento de correções orbitais
* Programação de manobras de atitude

Esse sistema é essencial para o planejamento das missões e manutenção adequada da posição orbital dos satélites.

---

## 📡 Coordenação das Estações Terrenas

O CCS também é responsável pelo gerenciamento operacional das Estações Terrenas.

Isso inclui:

* Otimização do rastreio de múltiplos satélites
* Resolução de conflitos de passagens simultâneas
* Coordenação das janelas de comunicação
* Planejamento operacional das estações

---

## ⚙️ Monitoramento da Missão

Entre outras atividades, o CCS também realiza o acompanhamento de:

* Consumo de propelente
* Sincronização do relógio de bordo
* Evolução da tensão das baterias
* Corrente do barramento elétrico
* Desempenho dos subsistemas do satélite

Essas atividades são fundamentais para garantir a confiabilidade e a vida útil das missões espaciais.
